| Услуга | Ориентир | Примечание |
|--------|----------|------------|
| Одна карбоновая деталь (custom) | $300–3k | Зависит от сложности, часов |
| Ремонт / усиление композита | $300–1k | |
| Interior trim panel | $500–2k | |

**Депозит:** 50%+ до заказа материалов. **Не** покупать на свои деньги.

---

## Продуктовая линейка (operational)

| Продукт | Реальная цена | Что входит (ориентир) |
|---------|---------------|------------------------|
| **Carbon / body package** | **$5k–20k** | Hood, trunk, splitter/diffuser, undertray, fenders (по мере готовности), paint prep/fitment |
| **Suspension / brake / chassis package** | **$15k–35k** | Coilovers, bushings, arms, subframe refresh, brakes, lines, alignment, wheels/tires (опционально) |
| **Interior package** | **$10k–30k** | Seats, door cards (composite), trim, steering wheel, hidden audio/CarPlay, wiring cleanup |
| **Engine / drivetrain package** | **$20k–60k+** | Rebuild M52/S52, cooling, exhaust (CARB-legal), LSD, trans refresh, engine bay detail |
| **Full E36 build** | **$90k–180k** | Все пакеты + интеграция, paint, QC, documentation, build book |

**Нижняя граница пакетов по отдельности:** ~$50k (без donor, без paint, без интеграции).  
**Full build** — не просто сумма пакетов: включает project management, fitment, paint, testing, margin.

## Hero / demo car (e36-00)

| | |
|---|---|
| **Назначение** | Brand prototype, портфолио, контент — не для продажи на старте |
| **Себестоимость (ориентир)** | **$60k–120k** |
| **Зависит от** | Состояние donor, объём своими руками vs outsource, paint/body, уровень интерьера и мотора |

Себестоимость demo ≠ цена для клиента. Demo — инвестиция в бренд; клиентский full build включает margin и повторяемость процесса.

## Как пакеты стыкуются

```
Carbon/body          $5k – $20k
Suspension/chassis   $15k – $35k
Interior             $10k – $30k
Engine/drivetrain    $20k – $60k+
─────────────────────────────────
Packages sum         $50k – $145k   (ориентир, без donor)

Full E36 build       $90k – $180k   (+ donor отдельно, типично $5k–25k)
```

Клиент может купить **один пакет** или **наращивать** по этапам — это и есть Stage 1 revenue до full commissions.

## Что влияет на позицию в диапазоне

| Фактор | Ниже ($) | Выше ($) |
|--------|----------|----------|
| Carbon | Hood + trunk only | Full aero, fenders, painted fitment |
| Chassis | Refresh + mid coilovers | Öhlins/MCS, BBK, forged wheels |
| Interior | Door cards + retrim | Custom seats, full rewire, premium audio |
| Engine | Refresh, stock power | Full rebuild, head work, dyno tune |
| Full build | Clean donor, минимум bodywork | Paint, structural repair, max spec |

## Не продаём на старте

- Commission full build **до** готового demo car и отработанных cost sheets
- Нелегальные engine swaps
- Полностью кастом без стандартизации пакетов
