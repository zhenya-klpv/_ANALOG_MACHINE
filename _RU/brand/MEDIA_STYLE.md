# Media Style — Quiet Premium Engineering

> Визуальный и тональный язык Analog Machine.  
> Не громко, не кричаще, не тиктоково — **спокойная уверенность**.

**Культурный референс:** **Phoebe Philo** — Old Céline (2008–2017), та же сдержанность в образе. См. [`BRAND.md`](BRAND.md#brand-dna--phoebe-philo).

## Celine test (перед публикацией)

Спросить перед каждым постом, роликом, фотосетом:

| Вопрос | Pass | Fail |
|----------|------|------|
| Это могло бы быть в lookbook Phoebe Philo? | Тихо, материалы, архитектура | Громко, лого, хаос |
| В кадре есть воздух? | Пространство, свет, один объект | Захламление, фильтры, наклейки |
| Шепчет или кричит? | Шепчет | Кричит |
| Лого / badge виден? | Маленький, заслуженный | Доминирует, рекламный |
| Color grade? | Нейтральный, stone, cream, black | Перенасыщение, неон, хром-блеск |

Если fail → переснять или переделать. **Лучше один тихий кадр, чем десять громких.**

## Visual language

- **Свет** — решает всё; карбон, металл, кожа без хорошего света = гараж
- **Чистота кадра** — даже обычная мастерская снимается графично, без мусора
- **Текстуры** — weave, stitch, machined edge, paint in sun
- **Пространство** — воздух в кадре, не clutter
- **Движение** — медленные push-in, slow motion деталей, rolling shots
- **Место** — утро, туман, Skyline, Highway 9, металл на верстаке

---

## Звук (недооценённый luxury layer)

- Щелчок трещотки, шлифовка, vacuum pump
- Закрытие двери, капот
- Холодный старт, idle, pull
- Шаги в мастерской, шуршание Alcantara
- Road ambience — шины, ветер, downshift
- Минимум музыки; ambient или silence

## Последовательность бренда

Один стиль на всё:

| Элемент | Правило |
|---------|---------|
| Цветокор | Один LUT / preset на весь проект |
| Темп монтажа | Медленный–средний; без jump-scare cuts |
| Титры | Минимум; один шрифт |
| Обложки YouTube | Одна система (layout + type) |
| Instagram grid | Чередование: hero / detail / process |
| Logo | В конце ролика, тихо |

## Рубрики (повторяемые форматы)

| Рубрика | Формат | Цель |
|---------|--------|------|
| **Before / After** | Carousel, 30 sec reel | Качество, доверие |
| **Why this part** | 60–90 sec talking + b-roll | Инженерность, экспертность |
| **On the bench** | Detail stills + macro video | Материалы, желание |
| **Build diary** | 5–12 min YouTube | Процесс, доверие |
| **Canyon morning** | Cinematic reel | Желание, бренд |
| **Weight & spec** | Static + number overlay | Ценность ($ justified) |
| **Founder note** | 1–2 min, direct camera | Философия, кто мы |

## Что контент продаёт

1. **Машину** — «Вот что мы можем построить»
2. **Детали** — «Вот что можно купить сейчас»
3. **Мастерскую / подход** — «Вот почему доверить проект нам»

Каждый пост должен явно или неявно вести к одному из трёх.

---

## Anti-patterns

- Грязный сервис в кадре без композиции
- Только wide shots без detail proof
- Только hero без process (нет доверия)
- Только process без hero (нет желания)
- Разный стиль от ролика к ролику
- Один красивый ролик раз в 4 месяца вместо ритма
