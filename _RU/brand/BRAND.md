# Brand — Analog Machine

> **Ядро бренда:** Analog Machine · **Handbuilt in California**

Три смысла в одной связке:

| Слово | Значение |
|-------|----------|
| **Analog** | Не digital appliance — механический объект |
| **Machine** | Настоящая машина, не гаджет |
| **Handbuilt in California** | Ремесло, место, культура — Highway 9, Skyline, Pacific Coast |

> Мы строим **аналоговые машины вручную в Калифорнии**.

Не «мы делаем BMW». Не «мы лучше BMW» — **наша интерпретация платформы**.

---

## Иерархия (финальная структура)

| Уровень | Значение |
|---------|----------|
| **Brand** | Analog Machine |
| **Studio / legal** | Analog Machine Works · Analog Machine Works LLC |
| **Cars** | AM36 · AM39 · AM Touring |
| **Badge** | AM |
| **Signature** | **Handbuilt in California** |
| **Philosophy** | Mechanical cars for the digital age |

### Для людей → **Analog Machine**  
### Для документов → **Analog Machine Works**

**Не использовать:** garage, customs, tuning, performance shop, **built by hands** (грамматика).

**На машине писать:**
- ✅ *Handbuilt in California*
- ✅ *Built by hand in California*
- ❌ *built by hands*

*Handbuilt* — Porsche / Singer / coachbuilt культура. Короче, дороже — идеально для шильдика.

## Rolex vs Apple Watch

Не «старое лучше нового». **Аналоговая вещь может быть дороже, желаннее, эмоциональнее цифровой.**

Rolex не обновляется по Wi-Fi — но желанный: механизм, статус, объект.  
Так же **Analog Machine** — feel, presence, motion — не feeds.
