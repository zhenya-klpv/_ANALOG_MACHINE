# Content Strategy — 90 Days (e36-00)

> Практический план на первые 3 месяца.  
> **Bootstrap mode:** фокус на **одной детали** (hood), не full build — см. [`../BOOTSTRAP.md`](../BOOTSTRAP.md).  
> Цель: **ритм + стиль + доверие** → первый paid inquiry.

Связано: [`CONTENT.md`](CONTENT.md) · [`MEDIA_STYLE.md`](MEDIA_STYLE.md) · [`../builds/e36-00/content/CONTENT_LOG.md`](../builds/e36-00/content/CONTENT_LOG.md)

---

## Ритм (держать стабильно)

| Канал | Частота |
|-------|---------|
| Stories | 3–4× / неделя |
| Reels | 2× / неделя |
| Feed post (carousel / still) | 1× / неделя |
| YouTube | 1× / 2–4 недели |
| Hero film | 1× на финал проекта |

> Лучше стабильно и в одном стиле, чем один шедевр раз в квартал.

## Month 1 — Foundation & Baseline

**Тема:** «Мы начинаем. Вот откуда.»

### Week 1–2: Donor story
| Deliverable | Тип | Канал |
|-------------|-----|-------|
| Cinematic walkaround (as-is) | Hero b-roll | Archive, teaser later |
| Inspection highlights | Reel 30–45 sec | IG |
| «Why this E36» founder clip | 60–90 sec | IG + YT Short |
| Baseline carousel (8–10 frames) | Before | IG feed |
| Mood board + LUT lock | Internal | Style bible |

### Week 3–4: Philosophy & setup
| Deliverable | Тип | Канал |
|-------------|-----|-------|
| «What we don't build» | Carousel / reel | IG |
| Workshop tour (clean angles) | Reel | IG |
| Build brief reveal (1 page visual) | Carousel | IG |
| YouTube Ep.1: **Project start + inspection** | 5–8 min | YouTube |

**Month 1 KPI:** 8 reels, 4 posts, 4 story weeks, 1 YT episode, style locked.

## Month 3 — Depth & Desire

**Тема:** «Машина оживает. Хочется сесть за руль.»

| Deliverable | Тип | Канал |
|-------------|-----|-------|
| First fitment on car | Reel | IG |
| Gap / surface detail macro | Carousel | IG |
| Interior material swatches | Reel | IG |
| «OEM+ vs tuning chaos» | Educational | YT Short / Reel |
| First start (if ready) | Reel | IG |
| Canyon scout B-roll (no final car yet OK) | Archive | Hero prep |
| YouTube Ep.3: **Chassis + first composite on car** | 6–12 min | YouTube |
| Founder: «California canyon car» | 2 min | IG + YT |

**Month 3 KPI:** audience understands **who / how / why**; 3 YT episodes total.

- [ ] Reveal film (60–120 sec)

- [ ] Driving film (2–5 min)
- [ ] Technical walkthrough (10–15 min YT)
- [ ] Philosophy film
- [ ] 10+ shorts из hero footage
- [ ] `_web/` launch с медиа

**Только после этого** — первые клиентские разговоры / packages.

## 5 задач контента — чеклист по месяцам

| Задача | M1 | M2 | M3 |
|--------|----|----|-----|
| 1. Качество (детали в кадре) | baseline | composites | fitment |
| 2. Доверие (процесс, честность) | inspection | layup hours | engineering choices |
| 3. Желание (атмосфера) | founder story | craft ASMR | materials / sound |
| 4. Ценность (куда деньги) | build scope | weight/spec | package hints |
| 5. Бренд (узнаваемый стиль) | LUT lock | consistent grid | tone established |
