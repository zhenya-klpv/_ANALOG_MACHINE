## Рекомендация

| Канал | Handle / URL | Статус |
|-------|--------------|--------|
| **Instagram** | **@analogmachineworks** | `@analogmachine` занят — не брать `_official` / `theanalogmachine` |
| **Website** | **analogmachine.co** | Целевой домен |
| **Legal / studio** | Analog Machine Works | Документы, footer |

**Правило:** коротко, чисто, без `_official`, `garage`, `customs`.

---

## Почему @analogmachineworks

- Solid, понятно: ателье, где строят — не просто философия
- Для старта даже **лучше**, чем голый @analogmachine
- На логотипах и в видео всё равно: **ANALOG MACHINE**

### Если занято — топ fallback

1. @analogmachines
2. @analogbuilt
3. @builtanalog
4. @analogcoachworks
5. @analogmotorworks

**Не брать:** analogmachine_ · theanalogmachineofficial · analog.machine (слабее как бренд)

### Отклонить / осторожно

| Handle | Заметка |
|--------|---------|
| @analogmachine.ca | `ca` читают как Canada |
| @analogworks.ca | То же |
| @amw.california | OK если качать AM badge; менее понятно без контекста |
| @analogmachine.co | Современно, но точка хуже для диктовки |
