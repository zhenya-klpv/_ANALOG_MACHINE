## Первый продукт — не капот

Капот дорог, долг, риск. **Первая победа за 2–4 недели:**

| Кандидат | Плюс |
|----------|------|
| Carbon intake duct | Видно в engine bay |
| Interior trim piece | Салон, тактильность |
| Rear seat delete panel | Простая геометрия |
| Small aero (splitter tip) | Motorsport vibe |
| Mirror cap | Маленькая, фотогеничная |
| Engine bay cover / bracket | Jewelry |
| **AM build plaque** | Бренд, почти без composite risk |
| Badge / center cap concept | Идентичность |
