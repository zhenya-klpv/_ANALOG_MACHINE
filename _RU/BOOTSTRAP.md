# Bootstrap Strategy — Zero Capital

> **Реальность:** нет свободных денег. Кредиты + аренда жилья.  
> **Главная цель:** не утонуть в кэше. Не «красивая мечта» — **выживание + умный рост**.

**Не начинаем как:** ателье, аренда barn, оборудование, сотрудники, полный рестомод.

**Начинаем как:** медиа + портфолио + маленькие paid-заказы на имеющихся ресурсах.

---

## Главное правило

> **Не брать новый финансовый риск, пока старые кредиты и аренда уже давят.**

Делать: **маленько, красиво, публично, без долга, с предоплатой.**

---

## Что НЕ делаем сейчас

| ❌ | Почему |
|----|--------|
| Аренда мастерской / barn | Фиксированный burn каждый месяц |
| Сотрудники | Payroll без cash flow |
| Крупные закупки на свои деньги | Материалы только с предоплатой клиента |
| Полный рестомод E36 | $60k–120k себестоимость — нереально сейчас |
| Мотор, подвеска, салон, paint, tuning сразу | Вне зоны силы + деньги |
| «Открываем бизнес» формально | LLC можно позже; сначала proof + cash |
| Искать VC / loan под мечту | См. `INVESTORS.md` — angels только после proof |

## Фазы (bootstrap)

### Phase 1 — Portfolio (0–3 мес) ← **СЕЙЧАС**

- Без аренды мастерской, без долгов, без крупных покупок
- Одна красивая деталь на E36 (капот — первый кандидат)
- Полный process content
- Inspection + baseline — бесплатно (уже есть машина)

**Cash out:** только материалы на одну деталь — минимизировать, покупать по мере нужды.

### Phase 2 — Первые платные работы (3–6 мес)

- Только маленькие заказы
- **Материалы + депозит на время — оплачены клиентом до старта**
- Не покупать материалы на свои деньги

### Phase 3 — Pre-order / deposit (6–12 мес)

- Если есть спрос на детали — **50–70% депозит** перед работой
- Стандартизировать 2–3 продукта (hood, splitter, door card)

### Phase 4 — Мастерская

- Только когда **аренда оплачивается работой**, не надеждой
- Или партнёр / инвестор с капиталом

### Phase 5+ — Long-term vision

Полные пакеты, AM36 builds, commissions — см. [`PRICING.md`](PRICING.md), [`VISION.md`](VISION.md).  
**Только после** стабильного cash flow и proof.

---

## Финансовые правила

1. **Никаких материалов на кредитку** без confirmed paid order
2. **Депозит 50–70%** перед любой клиентской работой
3. **Не масштабировать burn** (аренда, tools, hires) до повторяемых заказов
4. **Считать часы** — composite «сами сделаем» = 80+ ч легко
5. **Личный runway** — кредиты и аренда в приоритете над проектом
