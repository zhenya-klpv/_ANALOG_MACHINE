# Поставщики — сухой carbon (fabric)

> Wet layup + epoxy + vacuum bag. **Не prepreg** — см. [`PREPREG.md`](PREPREG.md).

---

## Spec для наших проектов

| Параметр | Target |
|----------|--------|
| Weave | **2x2 twill** (interior trim, conformability) |
| Tow | **3k** standard (OEM+ look) |
| Weight | **~200 g/m²** (6 oz) |
| Resin | Epoxy West 105/206 или equiv. |
| Finish | Satin 2K clear поверх |
