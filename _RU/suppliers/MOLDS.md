# Поставщики — формы (plug & female mold)

> Polyester mold + epoxy part (путь `_audi_s6`) · и epoxy mold (путь AM production).

---

## Два workflow

| | **Polyester mold** (FGS path) | **Epoxy mold** (production) |
|---|-------------------------------|----------------------------|
| Лицо формы | Orange **polyester** tooling gelcoat | Epoxy tooling surface coat |
| Тело | Iso tooling resin + mat/cloth | Epoxy + glass (no chop mat*) |
| Carbon part в форме | West 105/206 + release | West / epoxy + release |
| Кто продаёт | FGS, Fibre Glast, TAP | Fibre Glast, Rock West |
| Bootstrap | **★ рекомендован** | Позже |

*Chop strand mat — только polyester/vinyl ester, не epoxy.*

## Fiberglass Supply — [fiberglasssupply.com](https://fiberglasssupply.com)

| | |
|---|---|
| **Сильные стороны** | One-stop mold kit; West System; surf/small-shop DNA; хорошие how-to |
| **Слабые** | Shipping в CA дорогой; iso resin только gallon; нет PP sheet |
| **Цена** | $$ средний; **ship +$60–90** на малый заказ |
| **Когда** | Первый mold удалённо; повторный kit на новую деталь |

**Что брать:** Partall wax/PVA · orange tooling gelcoat · MEKP · iso resin · 1.5 oz mat · 6 oz cloth · West 105/206 · 406/403 · clay.

**Что не брать:** PP sheet → TAP; satin clear → paint shop.
