## Не skinning — настоящая деталь

**Отклонено:** carbon плёнка / skinning поверх OEM.

**Выбрано:**

```
OEM рамка (master) → форма (female mold) → carbon part
```

| Слой | Материал |
|------|----------|
| Лицевая часть | 2x2 twill carbon 200 g/m², epoxy, vacuum bag |
| Задняя сторона | **Гибрид:** carbon shell + OEM clips / 3D-print mounting features |
| Финиш | **Satin clear** (2K automotive) — OEM+, не супер-глянец |
| Толщина лицевой | ~1–1.5 mm; локальное усиление в креплениях |

**Не использовать:** forged carbon (не по эпохе C5), **cheap polyester для carbon layup**, silicone mold для этой задачи.

## Главная сложность

Красивая лицевая пластина — просто. **OEM посадка** — инженерия:

- внутренний проём под дефлектор
- тонкие кромки
- защёлки / крепления сзади
- зазоры с торпедой

**Правило:** сначала фото **задней стороны** OEM рамки → потом форма.

## Донорские детали

Купить **1–2 запасные рамки** A6/S6 C5 — не портить свою при первой форме.

После успеха — комплект в **одном weave-направлении:**
левый / центр / правый дефлекторы · дверные вставки · пепельница · рамка селектора.
