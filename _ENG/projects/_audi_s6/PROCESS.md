## Phase 1 — Master

On **donor** part (not the only one in the car):

1. Wash, degrease (IPA)
2. Fill scratches (light filler), sand P240+
3. Polish master — mold face quality depends on this
4. Check: clean edges, no chips

---

## Phase 2 — Female mold

**System (FGS #16141):** **polyester** mold · **epoxy** carbon part.  
Mold: Orange Tooling Gelcoat + iso resin + mat/cloth. Part: West 105/206 — see [`SHOPPING_LIST.md`](SHOPPING_LIST.md).

| Step | Action | Material |
|------|--------|----------|
| 1 | Parting line / flange | PP sheet (#11) + tooling clay |
| 2 | Fillets on master (pre-mold) | clay + epoxy paste (**406 + 403**) |
| 3 | Release on master | Partall wax 5–7 coats → PVA |
| 4 | Mold face ~20 mil wet | Orange Tooling Gelcoat + **MEKP** |
| 5 | First laminate layer | **6 oz cloth** (Style 7580) + iso resin + MEKP |
| 6 | Mold body stiffness | **1.5 oz chop mat** × 5–6 layers + iso resin + MEKP *(replaces 10 oz #7)* |
| 7 | Cure, demold master | — |
| 8 | Sand mold P240→P800 | inspect surface |

*Split mold only if undercuts block release — decide after back-side photos.*

## Phase 4 — Carbon part (production)

| Parameter | Value |
|-----------|-------|
| Fabric | 2x2 twill, 200 g/m² |
| Resin | West 105 + **206** (epoxy) |
| Release before layup | Wax + PVA on **polyester** mold face |
| Consolidation | **Vacuum bag** (film + fittings on hand) |
| Face thickness | ~1–1.5 mm |
| Back side | Hybrid: bond **OEM clips** from donor or 3D-print mounts |
| Clear | **Satin** 2K automotive |
| Sand | P800 → P1500 → P2000 → polish |
