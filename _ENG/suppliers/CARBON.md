# Suppliers — dry carbon (fabric)

> Wet layup + epoxy + vacuum bag. **Not prepreg** — see [`PREPREG.md`](PREPREG.md).

## Spec for our projects

| Parameter | Target |
|-----------|--------|
| Weave | **2x2 twill** (interior trim, conformability) |
| Tow | **3k** standard (OEM+ look) |
| Weight | **~200 g/m²** (6 oz) |
| Resin | Epoxy West 105/206 or equiv. |
| Finish | Satin 2K clear on top |
