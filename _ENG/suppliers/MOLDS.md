# Suppliers — molds (plug & female mold)

> Polyester mold + epoxy part (`_audi_s6` path) · and epoxy mold (AM production path).

---

## Two workflows

| | **Polyester mold** (FGS path) | **Epoxy mold** (production) |
|---|-------------------------------|----------------------------|
| Mold face | Orange **polyester** tooling gelcoat | Epoxy tooling surface coat |
| Body | Iso tooling resin + mat/cloth | Epoxy + glass (no chop mat*) |
| Carbon part in mold | West 105/206 + release | West / epoxy + release |
| Vendors | FGS, Fibre Glast, TAP | Fibre Glast, Rock West |
| Bootstrap | **★ recommended** | Later |

*Chop strand mat — polyester/vinyl ester only, not epoxy.*

---

## Fiberglass Supply — [fiberglasssupply.com](https://fiberglasssupply.com)

| | |
|---|---|
| **Strengths** | One-stop mold kit; West System; surf/small-shop DNA; good how-to |
| **Weaknesses** | Shipping to CA expensive; iso resin gallon only; no PP sheet |
| **Price** | $$ average; **ship +$60–90** on small order |
| **When** | First remote mold; repeat kit for new part |

**Buy:** Partall wax/PVA · orange tooling gelcoat · MEKP · iso resin · 1.5 oz mat · 6 oz cloth · West 105/206 · 406/403 · clay.

**Skip:** PP sheet → TAP; satin clear → paint shop.

## Fibre Glast — [fibreglast.com](https://www.fibreglast.com)

| | |
|---|---|
| **Strengths** | Best **learning center** guides; full mold range; Duratec primer; epoxy tooling options; 10 oz / 20 oz tooling fabric |
| **Weaknesses** | Ohio — shipping; not always cheaper than FGS |
| **Price** | $$–$$$ |
| **When** | Epoxy mold; quality primer; mold construction learning |

**Key items:** #90 iso resin · orange tooling gelcoat · PVA · parting wax · 10 oz style 7500 · Duratec gray surfacing primer · epoxy surface coat (carbon molds).

## TAP Plastics — [tapplastics.com](https://www.tapplastics.com)

**San Jose:** 1212 The Alameda · 408.292.8685

| | |
|---|---|
| **Strengths** | **$0 shipping** pickup; PP, HDPE, sign board; gelcoat; polyester; **TAP 314 epoxy** (West 105/205/206 analog); carbon section |
| **Weaknesses** | Not everything in stock; fewer mold-specific SKUs; incomplete West System range |
| **Price** | $$ local (without ship often **wins**) |
| **When** | #11 PP sheet; mat/cloth top-up; urgent; compare TAP 314 vs West |

**TAP 314 hardener map:** 314/102 ≈ West 205 fast · 314/109 ≈ 206 slow · 314/143 ≈ all-weather slow.

**Other Bay Area:** Dublin (HQ), San Francisco, Sacramento — see [`BAY_AREA.md`](BAY_AREA.md).
